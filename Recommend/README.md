# Movie Recommend
A movie recommendation web system

---
## DONE
* 用户管理 (登录, 登出, 注册)
* 电影信息展示
* 搜索电影 (按电影名搜索)
* 影院场次信息展示 (今明两天)
* 用户行为日志 (浏览记录)
